League of Legends Data Analysis Project

This project explores the League of Legends dataset to understand key metrics such as damage per minute (dmp) across different leagues and player positions.

Dataset Choice

Dataset: League of Legends ⌨️

Reason for Choice: I chose this dataset because I play this game and think it is interesting.

Hypothesis Testing

Null Hypothesis: There is no difference in the mean game damage per minute (dmp) between LPL and other tier one leagues (LCK, LEC, LCS).

Alternative Hypothesis: LPL has a higher mean game dmp compared to the other tier one leagues (LCK, LEC, LCS).

Test Statistic: Mean game dmp (damage per minute).

Prediction

Column to Predict: Player position.

Type of Problem: Classification.
